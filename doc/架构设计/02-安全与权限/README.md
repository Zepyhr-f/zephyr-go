## 维护规则
- 本目录承担跨端安全口径，前端、网关、后端均应回链到这里。
- 登录、Token、Header 透传、验签、DataScope 等主题变更时，必须先更新本目录 SSOT 文档。
- 技术分享中的补充说明不得覆盖本目录结论，只能做实现细节补充。
