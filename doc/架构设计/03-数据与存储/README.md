## 阅读建议
- 先看《数据模型总览》理解核心对象与关系边界。
- 涉及脚本执行、迁移策略时，再看《迁移与初始化方案》。
- 涉及 Redis key、TTL 和一致性策略时，查看《缓存策略设计》。

## 维护规则
- 本目录维护模型、迁移与缓存规则，不直接替代数据库脚本说明。
- 表结构或初始化顺序变更时，应同步检查数据库设计目录是否也需要更新。
- 业务字段语义以功能设计和建表脚本为准，本目录负责抽象层说明。
