## 1. 核心数据对象（建议维护）
- 平台域（RBAC）
  - 用户（User）
  - 角色（Role）
  - 菜单/资源（Menu/Resource）
  - 部门/组织（Dept/Org）
  - 字典（Dict）
- 业务域（示例）
  - 订单（Order）
  - 商品（Product）

---

## 2. 表关系（示例）
- User : Role = N : N（中间表 user_role）
- Role : Menu/Resource = N : N（中间表 role_menu / role_resource）
- Dept : User = 1 : N

> 具体 ER 图与建表脚本以 `04-数据库设计/` 为准；本文件只给“模型层面的解释与边界”。

---

## 3. 建议的命名与字段约定（摘要）
- 表：`sys_*`（平台域）、`biz_*`（业务域）
- 主键：`id`（bigint）
- 审计字段：`create_time/update_time/create_user/update_user/deleted`
